E-Commerce Order Engine

--->Project Overview
This project simulates a distributed e-commerce system with product management, cart handling, order processing, and payment simulation.

 --->Features Implemented
- Product Management
- Cart Management
- Order Placement
- Payment Simulation
- Low Stock Alerts
- Order Cancellation & Return
- Concurrency Simulation
- Failure Handling

--->Design Approach
The system is designed using layered architecture:
- Model Layer (POJO classes)
- Repository Layer (data storage)
- Service Layer (business logic)
- Controller Layer (CLI interface)

--->Assumptions
- Data is stored in memory (no database)
- Single-user CLI simulation
- Payment is randomly simulated

 --->How to Run
1. Open project in Eclipse/IntelliJ
2. Run `MainApp.java`
3. Use CLI menu to test features
